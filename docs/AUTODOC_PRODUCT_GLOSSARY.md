# AUTODOC: Product Glossary (Glosario de Producto)

Este documento unifica el lenguaje utilizado por el equipo técnico y los stakeholders de negocio, definiendo los conceptos clave del ecosistema Seller Command Center.

## 1. Glosario de Términos

| Término | Definición Conceptual | Contexto de Uso |
| :--- | :--- | :--- |
| **AI Advisor** | Asistente inteligente que analiza datos de ventas para sugerir acciones de optimización de inventario. | Sección destacada en el Dashboard principal. |
| **Envío a Full** | Servicio de logística integral donde el almacenamiento y envío es gestionado por el marketplace. | Recomendación principal del asesor inteligente. |
| **Glassmorphism**| Estilo visual de la interfaz que utiliza transparencias para dar sensación de profundidad. | Aplicado en tarjetas y paneles de información. |
| **Mercado Envíos**| Sistema logístico que asegura la entrega eficiente de los productos vendidos. | Métrica de eficiencia operativa. |
| **Platinium** | Nivel de reputación que indica alta calidad de servicio y volumen de ventas. | Indicador dentro de la métrica de Reputación. |
| **Seller Command Center** | Plataforma centralizada para la toma de decisiones estratégicas del vendedor. | Título general del proyecto. |
| **SKU** | Identificador único de un producto que permite gestionar el inventario de forma precisa. | Utilizado en las tarjetas de stock y alertas. |
| **Stock Crítico** | Estado de un producto cuyo inventario disponible está por debajo del nivel mínimo necesario. | Alerta prioritaria en el panel de control. |
| **Tendencia** | Indicador de crecimiento o decrecimiento en la demanda comparado con periodos anteriores. | Factor decisivo utilizado para generar recomendaciones. |

---

## 2. Resumen de Alineación

El uso consistente de estos términos asegura una comunicación eficiente. Al referirnos a un Insight o una Alerta, todos los miembros del equipo entienden tanto la implicación técnica como el impacto en la operación del negocio.
