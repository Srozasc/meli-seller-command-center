# AUTODOC: Product Glossary (Glosario de Producto)

Lenguaje común para alinear al equipo técnico y a los stakeholders de negocio.

## 1. Glosario de Términos

| Término | Definición | Contexto |
| :--- | :--- | :--- |
| **AI Advisor** | Sistema de asistencia inteligente para optimización de ventas. | Dashboard UI. |
| **Envío a Full** | Modelo logístico donde el marketplace gestiona el despacho final. | Estrategia de stock. |
| **Glassmorphism** | Estilo visual con transparencias y desenfoque profundo. | Diseño de interfaz. |
| **Run-rate** | Velocidad de venta diaria proyectada. | Cálculo de inventario. |
| **SKU** | Identificador único de un producto en el inventario. | Gestión de catálogo. |
| **Stock Crítico** | Alerta disparada cuando el inventario es insuficiente para cubrir la demanda. | Panel de alertas. |
| **Tendencia** | Crecimiento o caída porcentual de la demanda en un periodo. | AI Logic. |

## 2. Alineación de Conceptos
El uso de este glosario garantiza que conceptos técnicos (como un cálculo de tendencia en el código) se traduzcan directamente en el valor que el usuario percibe en la interfaz, eliminando vacíos de comunicación entre desarrollo y producto.
